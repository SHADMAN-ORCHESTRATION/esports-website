# Xevnex eSports 🎮

Xevnex eSports is a tournament platform concept built for mobile gamers in India.  
The idea is simple: make it easy for players to join ![Uploading image.png…]()

competitive matches, play with their squad, and actually win rewards.

This project is a frontend implementation of that idea, focused on creating a smooth and engaging user experience.

## 💡 About the Project

I wanted to build something that feels like a real product, not just a basic webpage.

Most tournament platforms are either too cluttered or confusing.  
Here, the goal was to keep everything straightforward:

- Land on the page  
- See the tournament details  
- Register quickly  
- Get ready to play  

That’s it.

---

## ✨ Features

The focus of this project is UI, interaction, and overall experience:

- Animated particle background using canvas  
- Smooth scroll and reveal animations  
- Countdown timer for tournaments  
- Interactive hero section with 3D tilt effect  
- Hover effects with glow transitions  
- Clean, modern eSports-style design  
- Fully responsive layout for mobile and desktop  

---

## 🎯 Games Supported
